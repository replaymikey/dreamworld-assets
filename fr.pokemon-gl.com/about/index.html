Pokémon GLOBAL LINK

Le Pokémon Global Link est un site Internet interactif avec Pokémon Version Noire et Pokémon Version Blanche.

Le Pokémon Dream World est le Monde des Rêves Pokémon qui s'étend dans le Pokémon Global Link. Il est possible de jouer à des mini-jeux avec des Pokémon qui ne sont pas originaires d'Unys, ou de communiquer avec d'autres joueurs.

Les utilisateurs possédant Pokémon Version Noire ou Pokémon Version Blanche peuvent rapporter les Pokémon avec lesquels ils sont devenus amis ou les objets qu'ils ont obtenus vers leur jeu.

Grâce à la Global Battle Union (GBU), les joueurs de tous les pays qui se sont affrontés via la CWF Nintendo peuvent voir leurs résultats affichés dans le Classement, et savoir quel est leur niveau. Il est également possible de s'inscrire aux Compétitions Wi-Fi.

Les résultats des Rencontres Aléatoires effectuées en mode Rang via la CWF Nintendo sont affichés dans le Classement, et permettent aux joueurs de savoir quel est leur niveau. Il est également possible de s'inscrire aux Compétitions Wi-Fi.

Et plein d'autres surprises sont à venir pour rendre votre expérience de Pokémon Version Noire et Pokémon Version Blanche encore plus inoubliable !

*Le PGL propose des contenus additionnels pour Pokémon Version Noire et Pokémon Version Blanche.
Ils n'ont aucune influence sur la progression dans le jeu en lui-même.
