Pour personnaliser son jeu

Depuis le Pokémon Global Link, vous pouvez télécharger des habillages pour le Pokédex et le C-Gear, ainsi que des programmes pour la Comédie Musicale Pokémon de Pokémon Version Noire et Pokémon Version Blanche.

Comment personnaliser votre version ①

Afin que votre personnalisation soit prise en compte dans votre jeu, vous devez lancer la Synchro-Jeu avant de vous connecter au Pokémon Global Link. Une fois l'habillage du C-Gear ou du Pokédex choisi, utilisez la Synchro-Jeu à nouveau pour réveiller votre Pokémon.

Comment personnaliser votre version ②

Pour personnaliser votre version, suivez les étapes ci-dessous :
1. Endormez un Pokémon dans Pokémon Version Noire ou Pokémon Version Blanche.
2. Connectez-vous au Pokémon Global Link, et dans "Personnaliser", choisissez les éléments que vous désirez changer, puis cliquez sur "Personnaliser votre version".
3. Utilisez la Synchro-Jeu dans Pokémon Version Noire ou Pokémon Version Blanche pour réveiller votre Pokémon.
4. Lorsque le Pokémon se réveillera dans votre console, les changements seront automatiquement pris en compte.

Si, après vous être connecté au Pokémon Global Link, vous avez joué dans le Pokémon Dream World, n'oubliez pas de cliquer sur "Quitter le Pokémon Dream World" avant d'utiliser la Synchro-Jeu.
