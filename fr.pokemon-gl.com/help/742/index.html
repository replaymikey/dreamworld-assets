Comment participer aux offres ?

Certaines offres requièrent un code cadeau ou un code spécial que vous pourrez obtenir grâce à certaines opérations spéciales du Club des Dresseurs Pokémon. Après vous être identifié sur le site du Club des Dresseurs Pokémon, cliquez sur le nom d'une offre dans la liste des offres, puis lisez bien l'onglet "Pour les personnes participantes" pour suivre les étapes qui vous permettront d'y participer.

Pour savoir comment participer, cliquez ici :
http://pglhelp.pokemon.com/fr/
