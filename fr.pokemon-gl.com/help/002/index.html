La version complète et la version d'essai

Quelle différence ?

La version complète

C'est la version dont vous pourrez profiter en enregistrant l'ID de Synchro-Jeu obtenu dans Pokémon Version Noire ou Pokémon Version Blanche.

La version d'essai

C'est la version que vous pourrez utiliser si vous ne possédez ni Pokémon Version Noire, ni Pokémon Version Blanche.

La version complète

En enregistrant l'ID de Synchro-Jeu que vous obtiendrez au cours de l'aventure dans Pokémon Version Noire ou Pokémon Version Blanche sur le site du Pokémon Global Link, vous pourrez profiter de contenus interactifs avec votre sauvegarde.
Dans le Pokémon Dream World, vous pourrez vivre les aventures dans le monde des rêves du Pokémon endormi dans votre carte de jeu, et même en ramener les objets qu'il aura trouvés, ou les nouveaux amis qu'il se sera faits.

De plus, vous pourrez voir votre Score de Rang et votre Classement dans la Global Battle Union.

La version d'essai

Même si vous ne possédez ni Pokémon Version Noire, ni Pokémon Version Blanche, vous pourrez visiter une partie du Pokémon Dream World, participer aux offres ou consulter la Global Battle Union.

Si plus tard vous obtenez Pokémon Version Noire ou Pokémon Version Blanche, vous pourrez, en enregistrant l'ID de Synchro-Jeu, passer à la version complète tout en conservant tout ce que vous aviez obtenu dans la version d'essai.
