Le Pokémon Global Link est un site Web qui fonctionne de concert avec les données sauvegardées des jeux pour consoles de la famille Nintendo DS™ Pokémon Version Noire et Pokémon Version Blanche.

Pour profiter du PGL, vous devez obligatoirement avoir un compte au Club des Dresseurs Pokémon.
