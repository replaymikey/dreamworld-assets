Si vous avez du mal à lire votre ID de Synchro-Jeu

● Pour profiter du PGL, vous avez besoin d'un Surnom PGL différent de celui de votre compte au Club des Dresseurs Pokémon.

● Le PGL utilisera les informations de sexe, anniversaire et lieu de résidence de votre compte au Club des Dresseurs Pokémon.

● Pour profiter des contenus du Pokémon Global Link en rapport avec les jeux vidéo Pokémon, il vous faudra effectuer des sauvegardes de vos jeux.

Pour profiter pleinement du Pokémon Global Link, il vous faut y enregistrer votre ID de Synchro-Jeu. Vous pourrez vérifier cet ID de Synchro-Jeu dans le menu principal, sous PARAMÈTRES SYNCHRO-JEU, une fois que vous aurez atteint un certain point dans l'aventure de Pokémon Version Noire ou Pokémon Version Blanche.

Vous ne pouvez associer à un compte du Club des Dresseurs Pokémon qu'un seul ID de Synchro-Jeu pour chaque version du jeu.

* Une fois l'ID de Synchro-Jeu enregistré, il ne pourra plus être changé.

Vous pouvez enregistrer un ID de Synchro-Jeu de Pokémon Version Noire et Pokémon Version Blanche en anglais, français, italien, allemand ou espagnol sur les sites Web du Pokémon Global Link en anglais, français, italien, allemand ou espagnol. Vous ne pouvez cependant pas enregistrer un ID de Synchro-Jeu de la version japonaise ou coréenne du jeu sur ces versions du Pokémon Global Link.

Vous jouez en ce moment à la version d'essai du Pokémon Global Link.

Vous pouvez jouer à la version d'essai du Pokémon Global Link même si vous ne possédez pas Pokémon Version Noire ou Pokémon Version Blanche. Vous aurez accès à d'autres contenus en enregistrant l'ID de Synchro-Jeu obtenu dans ces jeux sur le site du Pokémon Global Link. Vous recevrez votre ID de Synchro-Jeu grâce au C-Gear après avoir atteint la ville d'Ogoesse. Choisissez PARAMÈTRES SYNCHRO-JEU depuis le menu principal, ou bien INTERNET puis SYNCHRO-JEU depuis l'écran du C-Gear, et connectez-vous via la CWF Nintendo pour recevoir cet ID de Synchro-Jeu. Vous pourrez par la suite le consulter à tout moment dans PARAMÈTRES SYNCHRO-JEU depuis le menu principal. Vous pourrez enregister cet ID de Synchro-Jeu en allant dans votre profil, puis en cliquant sur Enregistrer ID de Synchro-Jeu. Veuillez endormir un Pokémon avant de procéder à cet enregistrement.
