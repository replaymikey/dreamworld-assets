            Qu'est-ce que la Synchro-Jeu ?

            Vous pourrez vérifier votre ID de Synchro-Jeu dans le menu principal une fois que vous aurez atteint un certain point dans l'aventure de Pokémon Version Noire, Pokémon Version Blanche, Pokémon Version Noire 2 ou Pokémon Version Blanche 2.

            Vous ne pouvez associer à un compte du Club des Dresseurs Pokémon qu'un seul ID de Synchro-Jeu pour chaque version du jeu.

            *Une fois un ID de Synchro-Jeu enregistré, il ne pourra plus être changé.

            Les informations suivantes seront enregistrées.
