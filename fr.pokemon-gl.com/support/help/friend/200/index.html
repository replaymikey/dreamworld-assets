Les Contacts

Il s'agit des utilisateurs de Pokémon Version Noire, Pokémon Version Blanche, Pokémon Version Noire 2 et Pokémon Version Blanche 2. Il y a 2 types de Contacts.

Les amis
Ce sont les personnes avec qui vous avez échangé votre code ami de Pokémon Version Noire, Pokémon Version Blanche, Pokémon Version Noire 2 ou Pokémon Version Blanche 2.

Les Copains d'échange
Ce sont les personnes avec qui vous avez échangé un Pokémon, par exemple via la GTS, dans Pokémon Version Noire, Pokémon Version Blanche, Pokémon Version Noire 2 ou Pokémon Version Blanche 2.

Vous pouvez échanger des Faciméls avec vos Contacts, ainsi que leur rendre visite dans le Pokémon Dream World.

Les Rêvamis peuvent se rendre visite dans leur maison.

*Pour se faire des Rêvamis, il faut avoir réglé le niveau de confidentialité de son profil sur "Public".
