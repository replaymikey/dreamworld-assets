Le Pokémon Global Link est un site Internet dont le contenu est directement lié aux sauvegardes des jeux Pokémon Version Noire, Pokémon Version Blanche, Pokémon Version Noire 2 et Pokémon Version Blanche 2.

Les Compétitions Wi-Fi sont des tournois de combats Pokémon organisés via la CWF Nintendo®. Vous pouvez vous y inscrire via le Pokémon Global Link et participer avec votre jeu.

Donn√©es

Il s'agit d'une page où vous pouvez consulter les divers classements obtenus à partir des données de jeu recueillies auprès de joueurs du monde entier.

Dans la section Compétitions Wi-Fi, vous pouvez voir les résultats des joueurs ou encore les Pokémon utilisés par les Dresseurs de haut niveau.

Grâce à la Synchro-Jeu du Bureau d'Enquêtes, les réponses des joueurs sont analysées et permettent de voir les particularités des Dresseurs par région du monde.
