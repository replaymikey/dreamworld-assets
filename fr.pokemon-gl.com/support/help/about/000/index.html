Qu'est-ce que le Pokémon Global Link ?

Le Pokémon Global Link

Le Pokémon Global Link est un site Internet dont le contenu est directement lié aux sauvegardes des jeux Pokémon Version Noire, Pokémon Version Blanche, Pokémon Version Noire 2 et Pokémon Version Blanche 2.

Pokémon Dream World

Le Pokémon Dream World est le Monde des Rêves, qui se trouve dans le Pokémon Global Link. Vous pouvez y changer la décoration de votre maison ou vous lier d'amitié avec des Pokémon grâce à des mini-jeux.

Compétitions Wi-Fi

Les Compétitions Wi-Fi sont des tournois de combats Pokémon organisés via la CWF Nintendo®. Vous pouvez vous y inscrire via le Pokémon Global Link et participer avec votre jeu.

Données

Il s'agit d'une page où vous pouvez consulter les divers classements obtenus à partir des données de jeu recueillies auprès de joueurs du monde entier.
Dans la section Global Trade Station (GTS), vous pouvez consulter par exemple le hit-parade des Pokémon les plus échangés, ou voir contre quels Pokémon un Pokémon particulier est échangé.

Dans la section Global Battle Union (GBU), vous pouvez consulter le Score de Rang et le Classement de tous les Dresseurs du monde qui se sont affrontés via la CWF Nintendo®, et ce pour chaque saison.

Dans la section Compétitions Wi-Fi, vous pouvez voir les résultats des joueurs ou encore les Pokémon utilisés par les Dresseurs de haut niveau.

Grâce à la Synchro-Jeu du Bureau d'Enquêtes, les réponses des joueurs sont analysées et permettent de voir les particularités des Dresseurs par région du monde.

Autres fonctionnalités

La fonction de personnalisation vous permet de télécharger des habillages pour le Pokédex et le C-Gear, ainsi que des programmes pour la Comédie Musicale Pokémon.

Avec les Faciméls, vous pouvez communiquer sous forme de messages en réponse à des questions, ou à compléter avec certains mots ; ils sont utilisables par tous pour une communication en toute sécurité.

Si vous échangez avec des joueurs parlant une autre langue, les mots que vous choisirez seront automatiquement traduits dans leur langue et ils pourront vous lire !

Pour profiter des contenus

Pour profiter des contenus du Pokémon Global Link liés à votre sauvegarde, il vous faut tout d'abord enregistrer un ID de Synchro-Jeu, que vous obtiendrez au cours de votre aventure dans le jeu vidéo.
Il vous faudra également utiliser la Synchro-Jeu pour endormir un Pokémon et vous identifier sur le Pokémon Global Link.
