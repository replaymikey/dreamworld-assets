ID para Sincronizar



Para poder disfrutar en Pokémon Global Link de los datos guardados con tu juego, deberás obtener un ID para Sincronizar de Pokémon Edición Negra o Pokémon Edición Blanca.

Después, acuesta a uno de tus Pokémon accediendo a SINCRO. JUEGO desde el menú EN LÍNEA de tu C-Gear e inicia sesión en Pokémon Global Link.

Para poder registrar el ID para Sincronizar en la versión en español de Pokémon Dream World, es necesario disponer de la versión en español de Pokémon Edición Negra o Pokémon Edición Blanca. El registro no podrá efectuarse con versiones extranjeras de Pokémon Edición Negra o Pokémon Edición Blanca.

Proceso de confirmación



Accede al servicio de la Conexión Wi-Fi de Nintendo® desde el menú de inicio de Pokémon Edición Negra o Pokémon Edición Blanca tocando CONFIG. SINCRONIZAR JUEGO. También puedes acceder desde el menú EN LÍNEA de tu C-Gear seleccionando SINCRO. JUEGO. Una vez que te hayas conectado, acuesta a uno de tus Pokémon y obtendrás un ID para Sincronizar.

Nota: Obtendrás el C-Gear cuando hayas logrado avanzar hasta Ciudad Gres en tu aventura.

Advertencias



Nota: Si no has utilizado nunca la función Sincronizar Juego para acostar a un Pokémon, no se reflejará en este sitio web la información de Pokémon Edición Negra o Pokémon Edición Blanca necesaria para poder sacarle todo el partido a los datos guardados. Si intentas registrar tu ID para Sincronizar en la versión en español de Pokémon Global Link sin haber acostado nunca a uno de tus Pokémon, solo podrás disfrutar de un registro temporal. Accede a la versión en español de Pokémon Global Link solo cuando hayas acostado a un Pokémon con la función Sincronizar Juego.
Del mismo modo, si registras un ID para Sincronizar desde tu perfil sin haber acostado antes a un Pokémon, el registro no surtirá efecto. Además, si estás usando un registro temporal, no podrás añadir un segundo ID para Sincronizar.
Nota: Ten en cuenta que, una vez que hayas registrado tu ID para Sincronizar, no podrás modificarlo ni eliminarlo.

Nota: El ID para Sincronizar consta de 10 caracteres. Las cifras 0 y 1 (cero y uno) y las letras O e I (o mayúscula e i mayúscula) no se utilizan. Presta atención para no confundir 8 con B (el ocho con la be), Y con V (la i griega con la uve), O con D (la o con la de) ni A con R (la a con la erre).
