Para disfrutar plenamente de estos contenidos, es necesario registrar un ID para sincronizar en Pokémon Global Link. Una vez que hayas avanzado lo suficiente en tu aventura, podrás seleccionar CONFIG. SINCRONIZAR JUEGO en el menú del título de tu juego Pokémon Edición Negra o Pokémon Edición Blanca para saber cuál es tu ID para Sincronizar.

Solo puede registrarse un ID para Sincronizar de Pokémon Edición Negra y uno de Pokémon Edición Blanca por cada nombre de usuario del Club de Entrenadores Pokémon.

Nota: Una vez registrado un ID para Sincronizar, este no podrá modificarse.

Puedes registrar un ID para Sincronizar de las versiones en inglés, francés, italiano, alemán o español de Pokémon Edición Negra y Pokémon Edición Blanca en los sitios web de Pokémon Global Link en inglés, francés, italiano, alemán o español. Sin embargo, no podrás registrar en Pokémon Global Link un ID para Sincronizar proveniente de la versión en japonés o en coreano.

Registro temporal en Pokémon Global Link.

La versión de prueba de Pokémon Global Link está disponible para aquellos usuarios que no tengan el juego Pokémon Edición Negra o Pokémon Edición Blanca para la consola Nintendo DS™. Podrás acceder a otros contenidos cuando registres en Pokémon Global Link el ID para Sincronizar que obtengas en Pokémon Edición Negra o Pokémon Edición Blanca. De esta manera, podrás disfrutar de los contenidos de tu juego en Pokémon Global Link.

Recibirás tu ID para Sincronizar una vez que hayas avanzado hasta Ciudad Gres en tu aventura y hayas obtenido el C-Gear en Pokémon Edición Negra o Pokémon Edición Blanca. Para consultar tu ID para Sincronizar, accede a la función Sincronizar Juego desde la pantalla EN LÍNEA o haz clic en CONFIG. SINCRO. JUEGO en el menú de inicio. Si quieres registrar tu ID para Sincronizar, ve a la página del perfil, haz clic en "Configurar perfil" y luego en "Registrar ID". Para efectuar el registro, es necesario que tu Pokémon esté acostado.
