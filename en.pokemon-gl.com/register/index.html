Understanding Game Sync IDs

● You need a PGL Nickname separate from the Pokémon Trainer Club screen name.

● The PGL will use the info you provided to register with the Pokémon Trainer Club.

● In the PGL, the saved data of the Game Card will be used to provide data linked with the Game Card.

Register your Game Sync ID to enjoy the PGL. You can check your ID from "GAME SYNC SETTINGS" at the main menu in Pokémon Black Version or Pokémon White Version once you've reached a certain point in your adventure.

One Pokémon Trainer Club account can have one Game Sync ID for both Pokémon Black Version and Pokémon White Version.

*The Game Sync ID cannot be changed once registered.

You can register Game Sync IDs of the English, French, Italian, German, or Spanish versions of Pokémon Black Version and Pokémon White Version at the English, French, Italian, German, or Spanish versions of the Pokémon Global Link. Please note that Game Sync IDs of Japanese or Korean versions can't be registered at the English, French, Italian, German, or Spanish versions of the Pokémon Global Link.
