Pokémon GLOBAL LINK

﻿

● Beim Pokémon Global Link benötigst du neben deinem Mitgliedsnamen vom Pokémon Trainer Club noch einen eigenen „PGL-Spitznamen“.

● Der PGL greift auf Daten zu, die du zum Zeitpunkt deiner Registrierung beim Pokémon Trainer Club angegeben hast.

● PGL-Inhalte, die mit deiner Pokémon-Spielesoftware kommunizieren, greifen auf die Daten des von dir benutzten Spieles zurück.

Um in den vollen Genuss der Inhalte zu kommen, musst du beim Pokémon Global Link eine Spielsynchro ID registrieren. Diese kannst du abrufen, indem du in Pokémon Schwarze Edition oder Pokémon Weiße Edition einen bestimmten Punkt deines Abenteuers erreichst und im Titelmenü die Option EINST. DER SPIELSYNCHRO-FUNKT. wählst.

Du kannst nur eine Spielsynchro ID und Mitgliedsnummer pro Spielesoftware Pokémon Schwarze Edition und Weiße Edition registrieren.

※Nach der Registrierung lässt sich die Spielsynchro ID nicht mehr ändern.

Du kannst eine Spielsynchro ID der englischen, französischen, italienischen oder spanischen Version von Pokémon Weiße Edition und Pokémon Schwarze Edition über den Pokémon Global Link registrieren. Bitte beachte, dass du keine Spielsynchro ID der japanischen oder koreanischen Version über den Pokémon Global Link registrieren kannst.
