            게임싱크 ID 코드는 「포켓몬스터 블랙」「포켓몬스터 화이트」 또는 「포켓몬스터 블랙 2」「포켓몬스터 화이트 2」에서 어느 정도 모험을 진행하면 타이틀 메뉴→「게임싱크 설정」에서 볼 수 있습니다.

            하나의 「포켓몬 오피셜 멤버십」 ID에 「포켓몬스터 블랙」「포켓몬스터 화이트」 「포켓몬스터 블랙 2」「포켓몬스터 화이트 2」 게임 소프트웨어 별로 하나의 게임싱크 ID 코드를 등록할 수 있습니다.
